Thesis

Title: Automating Large-Scale Hydrologic Mapping: A Scalable Machine Learning Framework for Hydrologic Feature Extraction
Advisor: Dr. Shaowen Wang
Abstract: This research develops transfer learning methods to improve spatial generalization of deep learning models for hydrologic feature delineation across physiographic regions using high-resolution remote sensing data.
